# Descripción del Sistema

**Versión:** 1.0  
**Fecha:** 19 de febrero de 2025

---

## Descripción General

Proyecto de pequeña escala desarrollado con **Flutter**, orientado a la integración y prueba de una API REST construida con **Node.js** y base de datos **MongoDB**. El sistema permite validar el flujo completo de comunicación entre el cliente móvil y el servidor backend, explorando operaciones básicas de consumo de datos, manejo de peticiones HTTP y persistencia de información en una base de datos NoSQL.
